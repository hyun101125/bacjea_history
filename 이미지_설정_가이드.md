## 🖼️ 이미지 파일 준비 요령

1. **파일명**: 위의 권장 파일명을 사용하거나, 원하는 이름을 사용한 후 `script.js`에서 경로만 맞춰주세요
2. **크기**: 한반도 지도(`hanbando.jpg`)와 동일한 크기 및 비율로 만드세요
3. **형식**: PNG (투명 배경 권장) 또는 JPG
4. **내용**: 해당 시대의 백제 영토 영역만 표시 (나머지는 투명)
5. **위치**: 프로젝트 폴더 루트에 저장하거나, `territory/` 폴더를 만들어서 저장 후 경로를 `'territory/territory-300.png'` 형식으로 입력

## ✅ 확인 방법

1. 이미지 파일들을 프로젝트 폴더에 저장
2. `script.js`에서 각 시대의 `territoryImage` 필드에 파일명 입력
3. 브라우저에서 `index.html` 열기
4. 타임라인 슬라이더를 움직여서 각 시대의 영토 이미지가 표시되는지 확인

## 💡 팁

- 모든 이미지를 한 번에 준비하지 못해도 됩니다. 일부 시대만 이미지를 넣고 나머지는 SVG로 표시됩니다.
- 이미지 파일명은 자유롭게 정할 수 있습니다. `script.js`의 경로만 맞춰주면 됩니다.
- 폴더 구조를 사용하려면: `'territory/18bc.png'` 형식으로 입력하세요.
